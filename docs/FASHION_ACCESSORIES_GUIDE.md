## 💎 推荐新增的高质感配饰

### 1. 胸针/徽章 ⭐⭐⭐⭐⭐（强烈推荐）
**为什么**：
- 小面积，AI 容易生成
- 可以和其他配饰叠加
- 极大提升奢华感

**提示词**：
```
"decorative crystal brooch pinned on the chest"
"vintage art deco brooch with gemstones"
"gold pin badge with pearl accents"
```

**最佳位置**：
- 西装翻领
- 领巾/丝巾上
- 帽子侧面

### 2. 珍珠耳环 ⭐⭐⭐⭐⭐
**为什么**：
- 经典优雅
- 与项链形成套装
- 适合白色/浅色宠物

**提示词**：
```
"elegant pearl drop earrings visible on both sides"
"diamond stud earrings"
"vintage chandelier earrings"
```

**注意**：需要 `prompt_strength >= 0.92` 才能生成

### 3. 领针/领带夹 ⭐⭐⭐⭐
**为什么**：
- 精致的细节
- 适合正式风格
- 容易与蝴蝶结/领带搭配

**提示词**：
```
"gold tie pin with pearl accent"
"decorative collar pin"
"vintage tie clip"
```

### 4. 怀表链 ⭐⭐⭐⭐⭐
**为什么**：
- 极具复古感
- 垂坠感增加层次
- 适合绅士风格

**提示词**：
```
"gold pocket watch chain draped from vest pocket"
"vintage watch fob with ornate chain"
```

### 5. 眼镜（非太阳镜）⭐⭐⭐⭐
**为什么**：
- 知性感强
- 适合学院风、文艺风

**提示词**：
```
"vintage round gold-rimmed reading glasses"
"monocle on delicate chain"
"cat-eye optical glasses"
```

## 🎯 配饰叠加策略（提升质感的秘诀）

### 原则：3-5 件配饰 = 最佳平衡

**✅ 推荐组合**：
```
主配饰（帽子/太阳镜）
+ 次配饰（项链/蝴蝶结）
+ 点睛配饰（胸针/耳环/领针）
```

**❌ 避免过度**：
- 不要超过 5 件配饰（会显得杂乱）
- 不要混搭冲突风格（如皇冠+街头帽）

## 🎨 提升质感的提示词技巧

### 1. 材质描述（关键！）
不要只说 "necklace"，要说：
```
"handcrafted beaded statement necklace with polished emerald stones"
"18k gold chain necklace with baroque pearl pendant"
"vintage art deco necklace with geometric crystals"
```

### 2. 奢华关键词
```
luxurious, exquisite, handcrafted, artisan, bespoke,
designer, couture, premium, high-end, sophisticated,
elegant, refined, opulent, ornate, delicate
```

### 3. 品牌暗示（不侵权）
```
✅ "Parisian haute couture style"
✅ "Italian luxury craftsmanship"
✅ "vintage Tiffany-inspired design"
❌ "Gucci necklace" (版权风险)
```
